Privacy Policy

Overview

This Privacy Policy explains how our Discord bot collects, uses, and protects your data.

Data We Collect

Our bot may collect and store the following information:

  • Guild Data: Server IDs and custom configuration settings
  • User IDs: For executing commands and logging
  • Channel IDs: For channel-specific features and settings
  • Command Usage Logs: Information about which commands are used

How We Use Data

We use collected data for:

  • Providing bot functionality and features
  • Improving performance and reliability
  • Debugging errors and issues
  • Storing user preferences and settings

Data Storage

Data is stored in a JSON configuration file on our servers. We implement reasonable security measures to protect this data.

Data Deletion

Users can request deletion of their data at any time. Simply remove the bot from your server to delete associated configuration data.

Third Parties

We do not sell, trade, or share your data with third parties. Data sharing only occurs as necessary to comply with Discord's Terms of Service.

Discord API

Our bot uses the Discord API and complies with Discord's Privacy Policy. Discord also collects data as outlined in their Privacy Policy.

Changes to This Policy

We reserve the right to update this Privacy Policy at any time. Users will be notified of significant changes.

Contact

For privacy questions or concerns, contact the server admins or bot developers.
